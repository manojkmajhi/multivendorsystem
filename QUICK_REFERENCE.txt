## ⚡ 5-Minute Setup

### 1️⃣ Enable Email (Supabase)
```
app.supabase.com
→ Authentication
→ Providers
→ Email: ON
```

### 2️⃣ Add Template (Supabase)
```
→ Authentication
→ Email Templates
→ One-Time Password
→ Edit HTML
→ Paste template (see below)
→ Save
```

### 3️⃣ Get API Keys (Supabase)
```
→ Project Settings (gear icon)
→ API
Copy:
  • SUPABASE_URL
  • SUPABASE_ANON_KEY
  • SUPABASE_SERVICE_ROLE_KEY
```

### 4️⃣ Update .env (Local)
```env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=eyJhbGci...
SUPABASE_SERVICE_ROLE_KEY=eyJhbGci...
APP_URL=http://localhost:3000
```

### 5️⃣ Restart Server
```bash
npm run dev
```

**Done!** Visit `http://localhost:3000/farmers/signup`

## 🧪 Test Commands

### PowerShell - Send OTP
```powershell
$body = @{ email = "test@cropsay.com" } | ConvertTo-Json
Invoke-WebRequest `
  -Uri "http://localhost:3000/api/farmers/send-otp" `
  -Method POST `
  -Headers @{"Content-Type"="application/json"} `
  -Body $body
```

### PowerShell - Verify OTP
```powershell
$body = @{ email = "test@cropsay.com"; otp = "123456" } | ConvertTo-Json
Invoke-WebRequest `
  -Uri "http://localhost:3000/api/farmers/verify-otp" `
  -Method POST `
  -Headers @{"Content-Type"="application/json"} `
  -Body $body
```

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| OTP not received | Check Email provider is ON in Supabase |
| | Check Email template is configured |
| | Restart server after .env changes |
| | Check spam/junk folder |
| OTP expired | OTP is valid for 15 minutes |
| | Request new OTP to get new code |
| OTP verification fails | Make sure code exactly matches email |
| | Remove any spaces |
| Server won't start | `npm install` and `npm run dev` |
| | Check port 3000 is available |
| Environment variables not loading | Restart server |
| | Check .env file format (no spaces around =) |

## 📊 OTP Flow

```
1. User submits email
   ↓
2. Server calls Supabase
   ↓
3. Supabase generates OTP
   ↓
4. Supabase renders email
   ↓
5. Supabase sends via SMTP
   ↓
6. User receives email
   ↓
7. User enters OTP code
   ↓
8. Server verifies with Supabase
   ↓
9. OTP verified ✓
   ↓
10. User completes profile
    ↓
11. Account created ✓
```
