# Multivendor System - E-commerce Platform

## Quick Start
```bash
npm install
npm run dev
```

## Structure
- `src/` - Application source code
- `public/` - Static assets
- `views/` - EJS templates
- `database/` - Database files
- `scripts/` - Build & deployment

## Environment
Copy `.env.example` to `.env` and configure your settings.

## Production
```bash
npm run build
npm start
```
