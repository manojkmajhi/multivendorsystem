==========================================

Troubleshooting:
  - Check Node.js version: node --version
  - Check port: netstat -ano | findstr :3000
  - View logs: pm2 logs allstrawhats
  - Clear cache: curl http://localhost:3000/__clear_cache

Common Issues:
  - Port in use: Change PORT in .env
  - DB connection: Check SUPABASE_URL
  - Images not loading: Check uploads directory
  - Performance: Enable PM2 cluster mode
