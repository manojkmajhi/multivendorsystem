## 🚀 QUICK START (2 Steps):

### Step 1: Run the SQL Code

1. Open file: `RUN_THIS_SQL.md` (in your project folder)
2. Copy ALL the SQL code
3. Go to: https://app.supabase.com
4. Click: **SQL Editor** → **New Query**
5. Paste and click **Run**
6. ✅ Done! You should see "Success. No rows returned"

### Step 2: Refresh Your Browser

1. Go to: http://localhost:3000/admin/
2. Hard refresh (Ctrl+Shift+R)
3. ✅ You should now see **Orders** (🛒) in the navigation!

## 🐛 If Orders Button Still Not Showing:

1. **Clear cache:** Ctrl+Shift+Delete
2. **Hard refresh:** Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
3. **Check SQL:** Make sure you ran the SQL in Supabase
4. **Restart server:** Stop and start `node server.js` again
5. **Check console:** Look for errors in browser console (F12)
