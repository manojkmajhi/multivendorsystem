## 🚀 How to Use

### Goto Supabase
```
1. app.supabase.com
2. Select your project
3. Authentication → Email Templates
4. Find "One-Time Password"
5. Click "Edit HTML"
```

### Copy & Paste
Copy the template from:
- `SIMPLE_OTP_TEMPLATE.md` (best reference)
- Or `QUICK_REFERENCE.txt`
- Or `OTP_QUICK_START.md`

### Save
```
Paste HTML → Save
```

## 📄 Template HTML

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: Arial, sans-serif; line-height: 1.6; color: #333; }
        .content { max-width: 500px; margin: 20px auto; padding: 20px; }
        .code { font-size: 36px; font-weight: bold; letter-spacing: 5px; text-align: center; padding: 20px; background: #f0f0f0; border-radius: 5px; font-family: monospace; margin: 20px 0; }
    </style>
</head>
<body>
    <div class="content">
        <p>Hi,</p>
        
        <p>Your OTP code is:</p>
        
        <div class="code">{{ .OTP }}</div>
        
        <p>Enter this code in the signup form. This code expires in 15 minutes.</p>
        
        <p>If you didn't request this code, ignore this email.</p>
    </div>
</body>
</html>
```

## 🔧 Customization

### Make OTP Bigger
```css
/* Change this: */
font-size: 36px;
/* To: */
font-size: 48px;
```

### Change Background Color
```css
/* Change this: */
background: #f0f0f0;
/* To: */
background: #e8f4f8;  /* light blue */
```

### Add More Space
```css
/* Change this: */
padding: 20px;
/* To: */
padding: 30px;
```
