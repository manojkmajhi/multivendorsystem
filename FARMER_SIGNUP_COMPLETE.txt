SIGNUP:

3. Gets OTP via Supabase Auth
4. Verifies OTP
5. Fills profile details + password

LOGIN:

2. Enter email + password
3. Redirects to dashboard

2. Enter email
3. Get OTP
4. Verify OTP
5. Set new password
