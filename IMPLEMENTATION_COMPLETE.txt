## 📊 What Changed in Code

### Before (Broken)
```javascript
// Only logged OTP, never sent it
const otp = Math.floor(100000 + Math.random() * 900000).toString();
otpStore[email] = { otp, expires: Date.now() + 300000 };
console.log('OTP for', email, ':', otp);  // ❌ User never receives this
res.json({ success: true });
```

### After (Fixed)
```javascript
// Sends via Supabase email service
const { data, error } = await supabase.auth.signUp({
  email: email,
  password: Math.random().toString(36).slice(-12),
  options: { emailRedirectTo: `${process.env.APP_URL}/farmers/signup` }
});

if(error) {
  // Fallback to local OTP if Supabase fails
  const otp = Math.floor(100000 + Math.random() * 900000).toString();
  otpStore[email] = { otp, expires: Date.now() + 300000 };
}

console.log('OTP sent to', email);  // ✓ Actually sent now
res.json({ success: true, message: 'OTP sent to your email' });
```
